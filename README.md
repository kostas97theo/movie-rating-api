# Movie Rating REST API

A secure RESTful API built with ASP.NET Core for managing movies and user ratings.

## Features
- ASP.NET Core Web API
- JWT Authentication & Role-based Authorization
- ASP.NET Identity
- Movies & Ratings CRUD operations
- Filtering, Sorting & Pagination
- Repository Pattern
- DTOs & AutoMapper
- Global Exception Handling Middleware
- Serilog Logging
- Swagger (JWT enabled)

## Tech Stack
- C#, ASP.NET Core
- Entity Framework Core
- SQL Server
- JWT
- AutoMapper
- Serilog
